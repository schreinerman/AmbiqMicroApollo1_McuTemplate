# AmbiqMicroApollo1_McuTemplate
MCU Template for Ambiq Micro Apollo 1 MCU

This is a project template for Ambiq Micro Apollo 1. It includes some
basic settings for e.g. Linker, C-Compiler.

Supported tool-chains: 
ARM        5.11
Atollic    4.3.1                     
emIDE      2.2
IAR        7.30
iSYSTEM    9.12.228
